# project mvp

Sports-betting prediction engine (repo: minimum-viable-predictions).

## 🚀 Quickstart

```bash
# 1. Clone the repo
git clone git@github.com:<your-org>/minimum-viable-predictions.git
cd minimum-viable-predictions

# 2. Create & activate virtualenv
python3 -m venv .venv
source .venv/bin/activate

# 3. Install in editable mode with dev tools
pip install -e .[dev]

# 4. Run the tests to verify setup
pytest
